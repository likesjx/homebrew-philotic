# Jaredlikes Philotic

## How do I install these formulae?

`brew install jaredlikes/philotic/<formula>`

Or `brew tap jaredlikes/philotic` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jaredlikes/philotic"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
